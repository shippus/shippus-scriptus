# How to use
Below, I'll document how to use each script.

### `dayNightCycle` script:
- Place in `ServerScriptService`.

*Otherwise*, the script should handle everything else by itself. Please report in the issues tab if otherwise.

**How to configure to your liking**
You only really care about the first lines *(Lines 11 to 13)*, basically the ones under the API setup, there you can change the value of duration_Day and duration_Night. The values can be anything *(As long as they're numbers)* and are translated into minutes.
